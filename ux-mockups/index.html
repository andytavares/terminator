Option C Workspaces as Windows High disruption

Each workspace is a native macOS window. Projects become top-level tabs within the window. Sessions are tab-bar entries below. The workspace rail vanishes — switch with ⌘` or the dock.

Option D Two-Level Tab Bar Medium disruption

Replace the three-level hierarchy (rail → panel → tab bar) with two horizontal tab bars: workspaces on top, sessions on bottom. Projects are a dropdown on each workspace tab. No sidebar.

Option E Quickswitcher Overlay Additive only

Keep the current sidebar exactly as-is. Add a ⌘K full-screen session picker (inspired by tmux's session picker and Linear's search). Shows recent sessions first, fuzzy matches across all levels.
